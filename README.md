# ⚽ DashboardFPL

A simple dashboard for **Fantasy Premier League** players using the **FPL API**.

## ✨ Features

✅ **Enter an FPL Team ID**  
✅ **Select a League** that the Team ID is part of  
✅ **View a clean dashboard** displaying:

📌 **Player Details**

- 🔹 Name
- 🔹 Team Name
- 🔹 League Name
- 🔹 Years Active

📌 **Performance Metrics**

- 📊 Gameweek Points
- 🏆 Overall Points
- 📍 League Position
