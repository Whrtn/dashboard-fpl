# DashboardFPL

A simple dashboard for Fantasy Premier League through the user of the FPL API

## Features

1. Entering an FPL Team ID
2. Selecting a league that the Team ID is in
3. A simple dashboard to display:

- Players name
- Years active
- Gameweek points
- Overall points
- Position in the league
